# iOS Safari Performance Baseline

Capture before/after metrics for the Astronomia Dragon configurator on iOS Safari emulation.

## Environment

- Viewport: 375×812
- UA: `Mozilla/5.0 (iPhone; CPU iPhone OS 17_4 like Mac OS X) AppleWebKit/605.1.15 (KHTML, like Gecko) Version/17.4 Mobile/15E148 Safari/604.1`
- Flow: load `/` → scroll to configurator → swap all 4 dragons × 3 metals (12 swaps) → tap View in AR

## Metrics script

Run in browser devtools / Playwright `page.evaluate`:

```js
({
  jsHeap: performance.memory?.usedJSHeapSize,
  canvases: document.querySelectorAll('canvas').length,
})
```
